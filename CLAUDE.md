# PalFO - Tic Tac Toe

## Project Structure
Single-file vanilla HTML/CSS/JS project: `tictactoe.html`. No build tools, package manager, or dependencies.

## Running
Open `tictactoe.html` directly in a browser.

## Architecture
- Array-based board state (9-element array)
- Priority-based AI strategy (win > block > center > corner > edge)
- Direct DOM manipulation (no framework)

## Theme
Dark theme: background `#1a1a2e`, accent `#e94560`, secondary `#0f3460`.
